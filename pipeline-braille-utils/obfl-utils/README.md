obfl-utils
==========

Building blocks related to [OBFL][] (Open Braille Formatting Language).

[library.xpl](src/main/resources/xml/library.xpl)
-------------------------------------------------

- `obfl:compare`: Compare two OBFL documents.
- `x:obfl-compare`: Compare two OBFL documents as a custom [XProcSpec][] assertion.

[OBFL]: http://code.google.com/p/obfl
[XProcSpec]: http://josteinaj.github.io/xprocspec
